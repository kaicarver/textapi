Kai's Tiny Text-posting API

gettext
Shows all the text posted so far.

pre-filled out text can be handy as a shortcut to post activities in a certain conventional, easy-to-mine format.








So far the API is very simple, and very experimental. You can post a bit of text with posttext, which takes a single parameter called... text, and you can retrieve all the bits of text posted so far with a GET from gettext (probably will use a single text endpoint for both in the final version).
